##How to convert image into base64 string

```
echo -n '<link rel="icon" href="data:image/png;base64,' > favico.txt; 
base64 yolo-icon-256x256.png | tr -d '\n' >> favico.txt; echo '">' >> 
favico.txt
```

The use it here:

```
    <link rel="icon" href="./static/images/yolo-icon-256x256.png">
```
